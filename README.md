# 🍿 Movie Ticket Transactional Chatbot

An intelligent conversational agent built using Natural Language Processing (NLP) designed to completely automate the movie ticket booking process. Beyond simple command-line scripts, this bot handles contextual dialogue, small-talk, entity extraction, and transactional state management.

## 🚀 Key Features
- **Stateful Transactions:** Guides the user through a multi-step booking funnel while retaining conversational context.
- **Entity Management:** Accurately extracts and manages names, dates, movie titles, and seat preferences.
- **Contextual Q&A:** Answers user queries flexibly, gracefully handling conversational drift and small-talk.

## 🛠️ Tech Stack
- Python
- NLP Libraries (NLTK / spaCy / Dialogflow)
- State Machine Logic
